#  Face Detection using Python

This project demonstrates a simple and effective implementation of **face detection** using Python. The model detects human faces in images using computer vision techniques.


# Features

* Detects faces in images
* Simple and beginner-friendly implementation
* Works in Google Colab environment


# Technologies Used

* Python 
* OpenCV
* NumPy
* Matplotlib
* Google Colab
